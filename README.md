本项目最初是为了备考证券从业资格证，但发觉证书对普通人实战意义有限后，果断**Pivot（转型）**！
现在的唯一目标是：通过系统性地阅读经典、观察市场、拆解数据，**建立属于我自己的大类资产（股票、债券、黄金）投资框架，看懂宏观经济的底牌，成为一个理性的实战投资者。**

## 🗺️ 学习地图与打卡清单 (The Real-World Financial Roadmap)

*本清单没有时间期限，你可以按照自己的节奏打卡。建议每完成一项，都在 GitHub 提交一次 Commit。*

### 🧠 模块一：认知重塑与财富思维 (Mindset & Psychology)
*投资市场的本质是反人性的。在投入真金白银前，必须先洗脑，建立正确的金钱观。*
- [ ] **纪录片**：观看《成为巴菲特》(Becoming Warren Buffett) -> *体会复利与长期的力量*
- [ ] **书籍阅读**：《金钱心理学》(摩根·豪泽尔) -> *理解为什么高智商的人也会在股市破产*
- [ ] **书籍阅读**：《富爸爸穷爸爸》 -> *彻底搞懂“资产”与“负债”的真实定义，建立现金流思维*
- [ ] **实战微操**：清点个人财务状况，算出一笔“绝不影响生活、亏光也不心疼”的闲钱，作为未来的实战本金。

### 🦅 模块二：宏观经济与世界的引擎 (Macroeconomics)
*搞懂利息、通胀与中央银行。这是投资股票和黄金的“地心引力”。*
- [ ] **神级视频**：在B站观看达利欧(Ray Dalio)的《经济机器是怎样运行的》(30分钟) -> *必须看懂！看懂信贷周期的扩张与收缩*
- [ ] **书籍阅读**：《周期》(霍华德·马克斯) -> *学习如何在经济过热时防守，在经济萧条时贪婪*
- [ ] **概念攻克 1**：搞懂“通货膨胀(CPI)”与资产贬值的关系。
- [ ] **概念攻克 2**：搞懂“美联储(Fed)加息/降息”对全球资金流动的影响。
- [ ] **实战微操**：下载并注册“华尔街见闻”或“金十数据”APP，连续一周观察每天推送的宏观经济数据。

### 📈 模块三：股市运转与指数基金 (Stock Market & ETFs)
*不炒妖股，不听内幕。学习如何通过买入优质资产，分享国家和时代的红利。*
- [ ] **纪录片**：观看央视纪录片《华尔街》(全10集) -> *了解资本市场的诞生与残酷*
- [ ] **书籍阅读**：《漫步华尔街》或《指数基金投资指南》(银行螺丝钉) -> *普通人战胜华尔街的最佳武器：宽基ETF*
- [ ] **概念攻克 1**：搞懂估值体系（市盈率PE、市净率PB、股息率ROE）。
- [ ] **概念攻克 2**：搞懂“宽基指数”（如沪深300、标普500、纳斯达克100）的编制规则。
- [ ] **实战微操 1**：在炒股软件搜索一家你最喜欢的公司（如腾讯/茅台/苹果），查看它的PE和过去5年的K线图。
- [ ] **实战微操 2**：在支付宝/天天基金网，模拟定投一只“沪深300ETF联接基金”，观察它的费用率(申购费/管理费)。

### 🥇 模块四：黄金、债券与大类资产配置 (Asset Allocation)
*防守赢下总冠军。当股市暴跌时，什么资产能保护你的财富？*
- [ ] **核心概念**：搞懂“无风险利率”（十年期国债收益率）为何是万物定价的锚。
- [ ] **黄金的逻辑**：总结黄金价格波动的核心驱动力（美元指数反向、实际利率、地缘避险）。
- [ ] **债券的逻辑**：搞懂“市场利率下降，债券价格上涨”的跷跷板原理。
- [ ] **实战微操 (终极测试)**：设计一个属于自己的“股债黄金平衡投资组合”（例如：60%股票ETF + 30%债券型基金 + 10%黄金ETF），并在模拟盘中建立这个组合。

## 🛠️ 我的投资工具箱 (My Arsenal)

1. **资讯与宏观**
   - [华尔街见闻](https://wallstreetcn.com/)：看全球宏观日历，盯美联储动态。
   - [雪球 App](https://xueqiu.com/)：看优质股民的深度研报（注意屏蔽情绪化喷子）。
2. **数据与分析**
   - [理杏仁](https://www.lixinger.com/)：查询A股/港股历史市盈率分位数的绝对神器（判断现在股市是贵还是便宜）。
   - [同花顺/东方财富]：看盘软件，建立自己的自选股观察池。
3. **交易渠道**
   - 股票/场内ETF：正规券商APP（如华泰、平安、中信等，要求万分之1.5以下的低佣金）。
   - 场外基金：支付宝 / 天天基金网。
